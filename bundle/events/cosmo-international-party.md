A recurring party hosted at [Second Home](/about.md) by the external Meetup group [Cosmo International Party Nagoya](https://www.meetup.com/ja-jp/cosmo-international-party-nagoya/). This is a separate, additional event from the [International Party](international-party.md) on Saturdays, which is run by [Global Friends in Nagoya](https://www.meetup.com/global-friends/) — for practical purposes, Second Home's own events and Global Friends events are the same thing.

# Schedule

Typically held Friday and Sunday evenings. Example dates from Meetup (July 2026):

* Sunday, July 12, 2026 — 18:00–21:00
* Friday, July 17, 2026 — 19:00–22:00
* Sunday, July 26, 2026 — 18:00–21:00

*Check [Cosmo International Party Nagoya on Meetup](https://www.meetup.com/ja-jp/cosmo-international-party-nagoya/events/) for the current schedule.*

# Entry

* Women: ¥1,000 (includes one drink)
* Men: ¥2,000 (includes one drink)
* Students: ¥500 discount with valid ID
* Additional drinks: ¥600+ (cash)
* Non-smoking; free dress code
