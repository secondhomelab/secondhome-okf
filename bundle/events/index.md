# Global Friends in Nagoya events

Events at Second Home are, for practical purposes, [Global Friends in Nagoya](/about.md#global-friends-in-nagoya) events. Each weekly slot below rotates through different themed nights (e.g. Burger Party, K-Pop Night) rather than always running under the same name — see each event's "Example Themes" section. **Any specific dates or theme names in these pages are illustrative only and go stale fast; check [Global Friends on Meetup](https://www.meetup.com/global-friends/events/) for the current schedule.**

* [International Party](international-party.md) - Recurring international-themed party, Saturday evenings 19:00–24:00.
* [International Midweek Mixer](midweek-mixer.md) - Weekly, Wednesday evenings 19:00–23:00.
* [Friday Night Meetup](friday-night-meetup.md) - Weekly, Friday evenings 19:00–24:00.
* [Quiz Night](quiz-night.md) - Monthly.
* [DJ Freestyle Meetup](dj-freestyle.md) - Monthly, Sundays.
